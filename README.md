DanmuPlayer
==================

Android端弹幕播放器
 
### Features:

- 支持播放BiliBili缓存的xml格式弹幕

- 支持弹幕屏蔽功能

- 支持局域网内获取弹幕播放

- 支持市面上流行的视频格式，包括RMVB
